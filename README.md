# Abstracci-n-y-Dise-o
Aplicación desarrollada en Python con Tkinter y Programación Orientada a Objetos (POO) para gestionar la nómina de empleados. Permite registrar datos, calcular el valor a pagar según el cargo y los días laborados, y generar reportes mediante una interfaz gráfica amigable. 🚀📋💼
