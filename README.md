## 🚀 Descripción del Proyecto

Esta es una aplicación de escritorio desarrollada en **Python** utilizando la librería **Tkinter** para el diseño de la interfaz gráfica de usuario (GUI). El sistema está estructurado bajo el paradigma de **Programación Orientada a Objetos (POO)**, aplicando conceptos clave como la **abstracción** y el **diseño modular** para lograr un código limpio, escalable y mantenible.

El software está diseñado para automatizar y optimizar la gestión de nómina de una organización, permitiendo:

* **📝 Registro Eficiente:** Captura y almacenamiento de datos esenciales de los empleados.
* **🧮 Cálculos Automatizados:** Procesamiento preciso del valor total a pagar basándose en el cargo del empleado y los días efectivamente laborados.
* **📊 Reportes Detallados:** Generación de resúmenes claros a través de una interfaz visual intuitiva, amigable y moderna.

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3
* **Interfaz Gráfica:** Tkinter
* **Paradigma:** Programación Orientada a Objetos (POO)
